// feature-section layout bench · 6 alternatives

Six interactive replacements for the bento.

Same six capabilities, Atlas, Recommendation & Auto Remediation, Autoscaler, Scheduling, Event Readiness, Showback, laid out six different ways. Every layout is working. Click, hover, swap, scroll. Pick one to ship.

// jump V1 · stepper V2 · loop V3 · radial V4 · explorer V5 · stack V6 · graph
// V1 · sequential stepper

Click 01 through 06. The story unfolds.

A horizontal numbered stepper at the top. Click any step or use the active one as a reading position. Most narrative-friendly, reads as "here's the loop, in order."

// V2 · continuous loop

Six nodes around one loop.

The "one continuous loop" claim made literal. All 6 capabilities sit on a circular path; click any to expand its details on the right. The loop diagram stays visible so the systemic relationship reads.

// V3 · hub & spoke

Atlas is the hub. Five spokes.

Atlas at the centre, every other capability runs on top of the topology Atlas provides. Click any spoke to expand its detail card. The hub stays put.

// V4 · dual-pane explorer

Like a docs explorer.

Left rail of capabilities, right pane of deep content. Tap any item, the right side updates. Closest analogue to the Stripe / Vercel docs UI, best for prospects who want to scan-then-deep-read.

// capabilities
// V5 · scroll-snap stack

One station per capability. Scroll through.

Each capability gets a wide horizontal station, number, copy, mini-visual. Vertical scroll-snap makes each one land cleanly. Most editorial-feeling option, no clicks required.

// V6 · connected graph

A graph, not a list.

Capabilities as nodes, real dependencies as edges. Click any node, its connections light up + the right panel shows the related capabilities by name. Most operator-grade view; reveals system relationships.
