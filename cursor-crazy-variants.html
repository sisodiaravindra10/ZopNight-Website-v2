// cursor variants · crazy edition · 12 JS-driven

Twelve cursors that actually do something.

Native CSS keywords were table-stakes. These are JS-driven cursors that follow the mouse, snap to elements, leave trails, stretch with velocity, draw beacons, and stamp the brand mark on click. Move the mouse inside any card to activate that variant.

// dot follower
X1 · DOT FOLLOWER
Single dot, no lag
Native cursor hidden, replaced with a 14px brand-orange square that follows the mouse perfectly. The simplest custom cursor.
// trailing dots
X2 · TRAILING DOTS
Six-dot fading tail
Each dot lerps to the position of the dot ahead of it · stationary cursor collapses to a single dot, fast cursor leaves a long fading trail. Reads as "cursor with momentum."
// magnetic snap · hover the targets
X3 · MAGNETIC SNAP
Ring snaps onto interactive elements
Default: 36×36 ink ring. Hover a `.target` and the ring resizes + repositions to wrap the target's bounding box. Brand-orange when snapped. Reads as "I know what's interactive."
// crosshair beacon
X4 · CROSSHAIR BEACON
Full-stage horizontal + vertical guides
Cursor is two thin orange lines that span the stage, intersecting at an ink dot. Same energy as a Figma comment-pin or a chart hover. Best for data / chart surfaces.
// inverse spotlight
X5 · INVERSE SPOTLIGHT
Everything dark except a window around the cursor
CSS radial mask reveals only a circular area around the cursor; the rest is filled ink. Wraps with a brand-orange ring. Use sparingly, feels like a magnifying glass on a redacted document.
// reactive · hover targets to morph
X6 · REACTIVE MORPH
Cursor changes shape per element
Default: 14×14 ink dot. Over a `.target`: 64×36 brand-orange pill. Smooth transition between states. Reads as "I'm context-aware."
// click anywhere to stamp
X7 · MARK STAMP
Click leaves a brand mark behind
Click anywhere and the ZopDev 4-quadrant mark stamps where you clicked, then fades over 600ms. The cursor itself is the orange dot. Most playful · works as a portfolio / agency cursor.
// live coordinates
x=0, y=0
X8 · COORDINATE DISPLAY
Cursor + live x,y readout
Cursor dot + a small mono chip floating bottom-right with live `x=…, y=…`. Reads as a CAD / dev-tool cursor. Best on canvas / map / chart surfaces.
// velocity stretch · move fast
X9 · VELOCITY STRETCH
Cursor stretches with motion
A 14×14 orange square that scales horizontally based on cursor velocity (and rotates to match the motion vector). Stationary: small square. Sprinting: stretched bar. Pure physics read.
// targeting reticle
X10 · TARGETING RETICLE
Big crosshair ring around the cursor
48px ink ring with stub crosshair lines + an orange centre dot. Most explicit "operator targeting" feel · use on the live-scanner playground or threat-radar surfaces.
// brand mark · 4-quadrant cursor
X11 · BRAND MARK CURSOR
The 4-quadrant ZopDev mark IS the cursor
20×20 brand-mark grid (blue / orange / · / green) follows the mouse. Strongest brand presence on hover-rich surfaces. Reserve for hero / signature pages, too much identity for app UI.
// trailing brand-color squares
X12 · TRAILING SQUARES
Three brand-color squares trail behind
Orange leads, blue follows ~80ms behind, green follows ~160ms behind. Reads as the brand mark "smearing" through the cursor's path. Most kinetic / playful option.
