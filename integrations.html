02 · cloud providers

Multi-cloud from day one.

Connect every major cloud provider in one session. ZopNight normalizes cost and resource data across clouds into a single unified view, no manual reconciliation.

AWS
5 minutes setup

Connects via read-only IAM role. Automatically discovers compute, storage, database, serverless, and network resources. Cost data pulled from AWS Cost Explorer API with tag-level granularity.

EC2RDSS3 LambdaEKSCloudFront ElastiCache+40 services
AWS deep dive →
Azure
5 minutes setup

Integrates via Azure Service Principal with Reader role. Covers VMs, managed databases, Kubernetes, App Service, serverless functions, and Cosmos DB. Cost data from Azure Cost Management API.

Virtual MachinesAzure SQL Blob StorageAKS App ServiceFunctions Cosmos DB
Azure solutions →
GCP
5 minutes setup

Connects via GCP Service Account with Viewer permissions. Covers Compute Engine, Cloud SQL, GKE, Cloud Run, BigQuery, and Pub/Sub. Billing data flows from Cloud Billing Export directly into ZopNight.
