01 · foundations

Colors.

Three brand colors, eleven cream-tinted neutrals. Never use pure #000 or #fff. Tint every neutral toward cream, it's what makes the brand feel like a single material.

Color strategy

Restrained (default · product) · neutrals + one accent ≤10%. Committed · one saturated color carries 30–60%. Drenched · the surface IS the color (heroes only). Pick before picking colors.

WCAG AA

Body text needs 4.5:1 minimum on its background. Use --g-600 or darker for 12–14px copy. --g-500 only for labels 14px+. --g-400 never as text on paper, it fails AA.

11 · motion

Motion.

Easing tokens are exponential ease-out variants. Never bounce, never elastic. Don't animate layout properties, use transform + opacity. Hover anywhere on a card to see the cards run.

--ease-out
cubic-bezier(.22, 1, .36, 1)
--ease-in-out
cubic-bezier(.65, 0, .35, 1)
--ease-snap
cubic-bezier(.19, 1, .22, 1)
--ease-spring
cubic-bezier(.2, .9, .3, 1.05)

Durations

--dur-fast · 160ms --dur-med · 250ms --dur-slow · 400ms

12 · don't

Banned patterns.

If you write any of these, rewrite the element with different structure. These are the AI-slop tells that ZopDev refuses to wear.

side-stripe borders

Coloured border-left > 1px on cards or callouts. Use full borders, background tints, or a leading number.

gradient text

background-clip: text + gradient. Decorative, never meaningful. Use a single solid color.

glassmorphism

Blurs and glass cards as default. Rare and purposeful, or nothing.

hero-metric template

Big number + small label + supporting stats + gradient accent. SaaS cliché.

identical card grids

Same-sized cards with icon + heading + text, repeated 5+. Vary sizes, asymmetric bento.

modal as first thought

Modals are usually laziness. Exhaust inline / progressive alternatives first.

em-dashes in copy

— or --. Use commas, colons, periods, parentheses.

bounce / elastic easing

Overshoot easing reads as gimmicky. Stick to exponential ease-out.

pure #000 / #fff

Tint every neutral toward cream. --paper + --ink, never raw black or white.

layout-property animation

Don't animate width/height/padding/margin. Use transform + opacity. For accordions, grid-template-rows: 0fr → 1fr.

italic text

The brand reads upright. Inline em tags stay for semantics; site-wide CSS forces font-style: normal.

round corners on cards

Square. Pills are 9999px only for genuine status indicators (badges).

13 · a11y

Accessibility checklist.

WCAG 2.1 AA minimum. Every interactive element has a focus state. Heading hierarchy never skips levels.

  • · 4.5:1 contrast for body text · 3:1 for 18px+ or 14px-bold large text
  • · Focus-visible state · 2px orange outline + 2px offset
  • · No h2 → h4 level skips
  • · Form inputs always paired with <label>
  • · Decorative SVGs have aria-hidden="true"
  • · Brand marks have aria-label or are inside a labeled link
  • · Every interactive element reachable via Tab
  • · prefers-reduced-motion · animations stop or fade only
  • · Test both html[data-theme="dark|light"]
  • · Test breakpoints at 375, 768, 1024, 1440
